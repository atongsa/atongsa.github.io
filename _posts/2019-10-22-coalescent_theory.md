## coalescent and phylogenetics
- role by trees
- model speciation
    - isolation model
        - species tree
            - gene trees
            - lineages will tend to coalesce within their species
            - can only coalesce with lineages from other species back in the ancestral species
- Molecular phylogenetics
    - species tree is assumed to exist
    - gene tree variables treated as a model parameter
    - all branches very long VS within-species coalescence times
        - coalescent can be ignored
        - is no need to sample more than one individual per species
        - recombination is completely irrelevant
        - Gene trees perfectly reflect species tree
        - repeated mutations erased phylogenetic information

### Population genetics
- genetic variation within a population
- Phylogenetics genetic variation between taxa
- assumed
  - Each taxon is represented by a single sequence
  - data for a single gene history for that gene
- Model
  - what happens within each population
  - within-population models to each population
- Assumptions; Wright-Fisher Model
  - Population of 2N gene copies
  - Discrete, non-overlapping generations of equal size
  - Parents are picked randomly
  - a specific parent for a gene 1/2N
